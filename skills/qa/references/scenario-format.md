# QA 시나리오 형식

Linear 코멘트에 기록할 때와 결과 리포트 형식.

## 시나리오 코멘트 (Linear 이슈에 기록)

```markdown
## QA 시나리오 (자동 생성)

### 시나리오 1: [제목]
- **URL**: [검수 대상 URL]
- **프로필**: [프로필명]
- **단계**:
  1. [URL]에 접속
  2. [요소]를 클릭/입력
  3. [기대 결과] 확인
- **기대 결과**: [성공 조건]
```

## 결과 리포트 (메인 세션에 반환)

```markdown
## QA 검수 결과

**이슈**: ZEP-181 - [제목]
**시간**: 2026-02-01 23:00

| # | 시나리오 | 결과 | 비고 |
|---|---------|------|------|
| 1 | [제목] | ✅/❌ | [상세] |

### 실패 상세 (있을 경우)
- **기대**: [기대 결과]
- **실제**: [실제 결과]
- **스크린샷**: [경로]
```
